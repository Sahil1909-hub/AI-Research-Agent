AI Research Agent

An AI-powered Research Agent built using LangChain, Groq LLMs, and Google Serper API for real-time web search and intelligent research responses.

This project allows users to ask research-oriented questions, and the agent searches the web, gathers information, and generates concise AI-powered answers.

Features
AI-powered research assistant
Real-time web search using Google Serper API
Tool calling support
Fast and lightweight
Modular and scalable architecture
Environment variable support with .env
